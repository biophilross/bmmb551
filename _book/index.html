This is an online book of all homework assignments for the class BMMB 551 as taught by Dr. Ross Hardison at Pennsylvania State University in the Spring of 2016. Instead of submitting static documents (.pdf, .docx, etc.) I decided to submit my homework assignments as webpages. This way content can be updated and I can even include interactive content using shiny.
